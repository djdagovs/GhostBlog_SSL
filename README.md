#Used for debian-based distributions

*Start with*
```
wget https://raw.githubusercontent.com/ianzhou233/ghost_blog_with_ssl/master/ghostwithssl.sh

sudo sh ghostwithssl.sh
```

2. Choose 'Y'
3. Enter your email address
